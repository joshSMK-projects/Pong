Project Name
    Pong

Project Description
    A recreation of the classic game Pong using JavaScript, CSS and HTML.

How to run the project
    1. Download the project
    2. Run the index.html file

How to use the project
    Upon running the index.html file press the "Start/Reset" button to begin or restart the game.
    Use W and S to control Player 1 (left side) and Up Arrow and Down Arrow to control Player 2 (right side).
    Aim for the opposing sides goal. Scores will update once anyone scores.
